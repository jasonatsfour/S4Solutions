# Odoo 13.0

Installation
=============
- Install the Application => Apps -> Edit LOT/Serial In MRP (Technical Name: edit_lot_mrp_base)


Edit LOT/Serial In MRP(Workorder)
===================================
- In Odoo standard flow, it doesn't allow us to edit lot/serial once we completed Work order. We have added functionality to manage edit Lot/serial even after completed Work Order.

- Once Manufacturing order is Marked as done we can not do any changes.
