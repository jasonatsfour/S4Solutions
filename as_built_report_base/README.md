# Odoo 13.0

Installation
=============
* Install the Application => Apps -> As-Built Traceability (Technical Name: as_built_report_base)


As-Built Traceability
==========================
* This module will supports Tracebility based on Manufacturing Order. It also include modification on existig Manufacturing Order using Odoo standard Repair Order Operations.


Steps
=====

1). Creating Manufacturing Order
------------------------------------------------------------
When we create Manufacturing Order , it display data as per default Tracebility Report.


2). To Modify Manufacturing Order Using Standard Repair Order
-----------------------------------------------------------------
To modify Manufacturing Order,  managing Repair order will be created for finished product(along with lot/serial) and Add or Remove Parts.

Based on Reapir Order Operation, system generate As Built Tracebility report.
